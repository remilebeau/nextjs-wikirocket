# WikiRocket

A Next.js SPA that searches Wikipedia for articles matching a given search query.

## [Click here to launch the app](https://nextjs-wikirocket-black.vercel.app/)

Created using

- Next.js
- Tailwind CSS
- TypeScript
- React
- Node.js
